📌 Nameservers cho
Hướng dẫn:
1. Đăng nhập vào nhà cung cấp tên miền của bạn.
2. Tìm phần quản lý DNS / Nameserver.
3. Thay thế nameserver hiện tại bằng các nameserver Cloudflare ở trên.
4. Lưu thay đổi và đợi DNS propagation (có thể mất 24-48 giờ).
5. Sau khi hoàn tất, trạng thái domain sẽ chuyển thành Active.

🔌 Kết nối Connector

💡 Connector là gì?
Connector (cloudflared) là chương trình chạy trên NAS/server, tạo đường hầm bảo mật giữa dịch vụ nội bộ và Cloudflare. Bạn cần chạy connector để tunnel hoạt động.
📱 Cách 1: Container Manager (dễ nhất)
1 Mở Container Manager trên DSM
2 Vào Registry → Tìm "cloudflare/cloudflared" → Download tag latest
3 Vào Container → Create
4 Chọn image cloudflare/cloudflared:latest
5 Mạng: chọn "Use the same network as Docker Host"
6 Phần Command, dán lệnh:
💻 Cách 2: SSH vào NAS
1 SSH vào NAS: ssh admin@[NAS-IP]
2 Chạy lệnh sudo -i (nhập mật khẩu admin)
3 Chạy lệnh Docker bên dưới:
📅 Cách 3: Task Scheduler (tự chạy khi khởi động)
1 Control Panel → Task Scheduler → Create → Triggered Task → User-defined script
2 User: root, Event: Boot-up
3 Tab Task Settings, dán lệnh Docker ở trên

➕ Thêm Route

Hostname đầy đủ sẽ là: ...
